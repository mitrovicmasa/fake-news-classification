# fake-news-classification

This project was developed as part of a course at the **Faculty of Mathematics, University of Belgrade.

The goal is to classify news articles as "real" or "fake" using various techniques.

## 📊 Datasource
The dataset used in this project is the [Fake News Classification Dataset](https://www.kaggle.com/datasets/saurabhshahane/fake-news-classification) from Kaggle.

## 🤖 Models Implemented
The following models were implemented and compared:

* **Multinomial Naive Bayes (NB)**
* **Logistic Regression**
* **Linear Support Vector Classification (LinearSVC)**
* **Passive-Aggressive Classifier (PA)**
* **Light Gradient Boosting Machine (LGBM)**
* **Long Short-Term Memory (LSTM)** - Neural Network
* **K-Means** - Unsupervised Clustering
